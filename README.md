# next-unocss

Next.js with unocss.

## Installation

```shell
npm i -g pnpm
```

```shell
pnpm i
```

## Start Next.js with UnoCSS by using unocss-cli

```shell
pnpm dev:cli
```

## Start Next.js with UnoCSS by using @unocss/webpack

```shell
pnpm dev:webpack
```
